# sbog/imgproxy

Role to deploy imgproxy for image resizing

## Requirements

Ansible 2.8+

## Role Variables

Can be seen in [defaults](`defaults/main.yml`)

## Dependencies

None

## Example Playbook

```yaml
    - hosts: servers
      roles:
         - { role: imgproxy }
```
